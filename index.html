4x4x4 LED Cube

Rendering "3D Images" using nothing but LEDs.

Using the Arduino, a protoboard, a few transistors and resistors, I designed a cube consisting of an array of LEDs that are individually addressable.The setup allows the animation of monochromatic images in 3D space. The cube is controlled by an Atmel MCU which toggles the gates of the drive transistors which control each layer within the cube. A fast refresh rate was used in order to utilize effects of the persistence of vision. The purpose of this project was to gain experience in embedded system design.

Supercapacitor Charger

Charging capacitors via the Air.

Utilizing TI's BQTesla series of Inductive Charging ICs, I developed the circuitry required to demonstrate Wirelessly charging capacitors as big as 20 Farads within a matter of 60 to 90 seconds. My design setup involved using an Inductive coil transmitter, a receiver and a 20F capacitor bank.
